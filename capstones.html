Ten portfolio projects (named course combinations)
01
Document intake portal
Technologies: PHP, MySQL, file uploads, Python batch (parsing / features), Pandas + scikit-learn classifier or scoring.
Deliverable: Upload docs → rows in MySQL → Python enriches with category/score → PHP dashboard lists results.
Combine: Projects in PHP and MySQL (Eduonix) · Automate the Boring Stuff (Sweigart) · Python for Data Science and Machine Learning Bootcamp (Portilla)
02
Smart ticket / intake UI + Python API
Technologies: React, Flask or FastAPI, JSON, Pandas + sklearn, simple explainability (e.g. top features).
Deliverable: React form → /predict → show label, confidence, short explanation string.
Combine: React Mega Course (Sufa Digital) · The Python Mega Course (Sulce) · Portilla data science bootcamp
03
Full stack: PHP auth + React UI + Python inference
Technologies: PHP sessions/auth, MySQL, React SPA, REST between tiers, Python only for model.
Deliverable: PHP owns login + metadata; React is product face; Python service is isolated inference — clean interview boundary.
Combine: PHP From Scratch (Traversy) · Projects in PHP and MySQL (Eduonix) · React Mega Course (Sufa) · Python Mega Course (Sulce)
04
Custom dataset deep learning + portfolio write-up
Technologies: PyTorch training loop, CNN or strong tabular model, metrics/plots, README in “40 projects” style.
Deliverable: One custom dataset end-to-end: EDA → train → evaluate → saved weights + figures.
Combine: PyTorch for Deep Learning Bootcamp (Neagoie / Bourke) · Portilla · 40 Real World Data Science, Machine Learning Projects 2025 (Pianalytix)
05
Automation-first ML artifact pipeline (AWS)
Technologies: Scheduled Python, training script outputs artifacts, S3, optional batch or metrics file, notifications.
Deliverable: Nightly: new CSV from watched folder → refresh model or batch scores → push metrics to S3 (+ optional email).
Combine: Automate the Boring Stuff (Sweigart) · Portilla · AWS MLA-C01 (Schuler)
06
RAG over your document folder
Technologies: Chunking/cleaning (Sweigart), embeddings + vector store + tool use from agent courses, optional React chat UI.
Deliverable: Local PDF/Markdown folder → indexed → query with citations (CLI first is fine).
Combine: Intro to AI Agents and Agentic AI (365 Careers) · Agentic Track / MCP (Donner, Ligency) · Automate the Boring Stuff · optional React Mega Course
07
Bio pipeline CLI → scoring model
Technologies: Biopython / Entrez / parsing (Mariano), biology context (Siddique), features → sklearn or PyTorch head, plots.
Deliverable: FASTA or accession in → fetch/parse → features → prediction + simple report figure.
Combine: Introduction to programming for Bioinformatics with Python (Mariano) · Applied Bioinformatics (Siddique) · Portilla or PyTorch bootcamp
08
Same model, AWS-hosted inference
Technologies: S3 artifacts, SageMaker training or hosting, IAM, thin React or PHP client.
Deliverable: Take project 04 (or similar) and deploy inference the AWS ML course way.
Combine: AWS MLA-C01 (Schuler) · PyTorch for Deep Learning Bootcamp · React Mega Course or PHP / MySQL projects
09
Mega Course app + one ML feature
Technologies: App structure from Mega Course, beginner clarity from Intro Python, one Portilla-style model (recommendation, forecast, classifier).
Deliverable: Extend one “10 apps” style project with a single shipped ML feature + tests.
Combine: The Python Mega Course (Sulce) · Introduction To Python Programming (Jain) · Portilla
10
Portfolio site that tells the stack story
Technologies: React case-study pages, optional PHP for blog routing, three repo deep-links with stack diagrams.
Deliverable: Public site linking GitHub; each case study maps courses → tech → outcome.
Combine: React Mega Course · PHP From Scratch (optional static/server routing) · narratives shaped like 40 Projects (Pianalytix)

Suggested focus: ship #2 then #3 for web + ML; add #6 for AI engineer narrative; use #7 + #8 for research + cloud.
