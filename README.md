# Career Threshold Tracker

Static HTML/CSS/JS toolkit: a **course-progress tracker** (sliders, roles, salary bands, career paths), a **100-project Engineering Journal** (5 stacks, 4 tiers, project-level LLM prompts), and **Udemy capstone** helpers.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Contents

| Area | Entry |
| ------ | -------- |
| Site map | `index.html` |
| Tracker | `career_threshold_tracker.html` |
| 100 projects | `curriculum.html`, `projects.config.json`, `portfolio_projects.js` |
| Capstones + prompts | `capstones.html` |
| Shared styles | `site.css` |
| Legacy curriculum data | `compound_curriculum.js` |
| Course prompts | `course_prompts.js` |

Progress in the tracker is saved in **localStorage** in your browser.
