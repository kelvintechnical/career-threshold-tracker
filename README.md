# Career Threshold Tracker

Static HTML/CSS/JS toolkit: a **course-progress tracker** (sliders, roles, salary bands, career paths), **160 compounding curriculum projects** with copy-paste LLM prompts across eight tier pages, and **Udemy capstone** helpers.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Contents

| Area | Entry |
|------|--------|
| Site map | `index.html` |
| Tracker | `career_threshold_tracker.html` |
| 160 projects | `curriculum.html`, `tier-1.html` … `tier-8.html` |
| Capstones + prompts | `capstones.html` |
| Shared styles | `site.css` |
| Curriculum data | `compound_curriculum.js` |
| Course prompts | `course_prompts.js` |

Progress in the tracker is saved in **localStorage** in your browser.
